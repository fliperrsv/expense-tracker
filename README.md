# 💰 Финансовый трекер (Expense Tracker)

[![GitHub Pages](https://img.shields.io/badge/Демо-expense--tracker-ff6b4a?style=flat-square&logo=github)](https://fliperrsv.github.io/expense-tracker/)

**Приложение для учёта доходов и расходов** с графиком расходов по категориям, фильтрацией, тёмной темой и сохранением данных в браузере. Реализовано на Vue 3 (Composition API).

🔗 **Живое демо:** [https://fliperrsv.github.io/expense-tracker/](https://fliperrsv.github.io/expense-tracker/)

---

## 📦 Функционал

- ✅ Добавление доходов и расходов с категориями
- ✅ Автоматический расчёт баланса, общей суммы доходов и расходов
- ✅ Круговая диаграмма расходов по категориям (Chart.js)
- ✅ Фильтрация операций: все / доходы / расходы
- ✅ Тёмная / светлая тема (запоминается в `localStorage`)
- ✅ Сохранение всех транзакций в `localStorage`
- ✅ Адаптивный дизайн для мобильных устройств
- ✅ Написано на **Vue 3 (Composition API)**

---

## 🛠 Технологии

- **Vue 3** – реактивный интерфейс
- **Chart.js** – визуализация расходов
- **CSS3** – тёмная тема, адаптив, переменные
- **JavaScript (ES6+)** – логика, работа с localStorage
- **Font Awesome** – иконки

---

## 🚀 Как запустить локально

1. Склонируйте репозиторий:
   ```bash
   git clone https://github.com/fliperrsv/expense-tracker.git
   cd expense-tracker
```

2. Откройте файл index.html в любом современном браузере.
3. Всё готово! Данные сохраняются в локальном хранилище.
